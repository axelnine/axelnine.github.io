Experience

Data Analytics Intern

Exponentia Datalabs, Mumbai

My role at Exponentia Datalabs as an intern was to build a sentiment analysis model based on Recurrent Neural Networks specifically LSTMs in order to improve customer analytics for clients. The clients were mainly from the finance domain.

Some highlights of my internship were as follows:

- Migrated the sentiment analysis component of the companys product using machine learning from Support Vector Machines to Recurrent Neural Networks (RNNs). Improved the accuracy of the model from 78% to 86%.

- Further improved the accuracy by developing a phrase chunking component to extract sentiment aspects. Further improved the accuracy and precision by approximately 2%. Acted as an integral component to extract multiple sentiments from a sentence.

- Reduced training time by 10% by optimizing different hyperparameters for RNNs in python using the keras deep learning library.

- Connected database through MySQL connector to give real-time updates on social media data.
